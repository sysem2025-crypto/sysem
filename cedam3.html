Cedam 3

Sistema di automazione e telecontrollo per impianti di sollevamento acque

Panoramica

Piattaforma software per il telecontrollo di impianti idrici: gestione serbatoi, automazione pompe, allarmi configurabili e diagnostica da remoto.

  • Due modalità operative: Automatica (controllo a volumi) e a Livelli serbatoio
  • Automazione gruppi pompa con timer minimi partenza/esercizio
  • Programmazione fasce orarie con soglie differenziate
  • Allarmi configurabili con soglie, isteresi e ritardi
  • Logger eventi e storico allarmi

Architettura

Applicazione Windows nativa con database SQL Server per configurazione e storicità dati.

  • Frontend: interfaccia nativa Windows MFC (C++)
  • Backend: SQL Server su rete locale
  • Comunicazione: IP e Radio verso RTU in campo
  • Statistiche: connettività IP/Radio per ogni RTU

Impianti gestiti

Controllo simultaneo di più sistemi con logiche indipendenti.

  • S.Anna — 3 serbatoi, automazione a volumi/livelli
  • Piardarossa — 1 serbatoio, automazione a volumi/livelli
  • Frassineto — monitoraggio e statistiche
  • Terranova — monitoraggio e statistiche

Programmazione Fasce Orarie

Il sistema modula le soglie di volume dei serbatoi su base oraria, con valori differenziati per feriale, sabato e festivo.

  • Tabella livellifasceorarie_sanna / livellifasceorarie_piardarossa con soglie 0-23 per Feriale, Sabato, Festivi
  • Interpolazione lineare tra una soglia oraria e la successiva per transizioni graduali
  • Calcolo portata richiesta: Q = (soglia_corrente - volume_attuale) / tempo_recupero
  • Accensione/spegnimento pompe per inseguire il profilo target con isteresi

Specifiche tecniche

  • Controllo fino a 5 serbatoi e 100 pompe raggruppabili
  • Sistema allarmi con soglie, isteresi, ritardi e tacitazione
  • Controllavaggio programmato
  • Fasce orarie personalizzabili
  • Watchdog di sistema e diagnostica comunicazioni
