    Famiglie Microcontroller 8-bit

    Atmel (Microchip) AVR

    Serie: ATmega (ATmega328P, ATmega2560), ATtiny (ATtiny85, ATtiny441), AT86RF (wireless)

    Caratteristiche: 8/16-bit RISC, 2-128KB Flash, periferiche USART, SPI, I2C, ADC, PWM

    Applicazioni: hobby, prototipi, IoT, wireless sensor networks

    Popolarità: molto alta, ampio ecosistema, facile programmazione con AVR-GCC/Arduino

    Microchip PIC10/12/16/18

    Serie: PIC10F322, PIC12F1822, PIC16F18877, PIC18F26K22

    Caratteristiche: ultra-basso consumo, 1-64KB Flash, periferiche di base, bootloader integrato

    Applicazioni: dispositivi a basso consumo, sensori embedded, controllo di potenza

    Popolarità: molto diffusa in applicazioni industriali e di consumo

    STM8 (STMicroelectronics)

    Serie: STM8S, STM8L, STM8A, STM8AF

    Caratteristiche: 8-bit CISC, 16-128KB Flash, periferiche complete (CAN, USB, Ethernet), sicurezza

    Applicazioni: automotive, industriale, medicale

    Popolarità: crescente, supporto MDK-ARM, tool di sviluppo gratuiti

    Intel (ex-Dallas) 8051/8052

    Serie: 8051, 8052, 89051, DS89251

    Caratteristiche: architettura classica, 4-64KB ROM, periferiche seriali, timer, interrupt

    Applicazioni: controllo industriale, sistemi embedded legacy, emulazione

    Popolarità: storico, ampio supporto software, ancora usato in molti sistemi embedded

    Renesas R8C

    Serie: R8C/1, R8C/2, R8C/3, R8C/4

    Caratteristiche: microcontroller automobilistici, sicurezza funzionale integrata, CAN, LIN, FlexRay

    Applicazioni: automotive, controllo motore, body control modules

    Popolarità: specializzato, focus su automotive safety

    NXP (ex-Freescale) 8-bit

    Serie: S08, S12, S12X, S32

    Caratteristiche: ampia gamma, da entry-level a mid-range, periferiche complete

    Applicazioni: controllo industriale, automazione, dispositivi medici

    Popolarità: molto diffusa, soprattutto in applicazioni industriali

    TI MSP430

    Serie: MSP430F, MSP430G, MSP430X

    Caratteristiche: ultra-basso consumo (<10µA @ standby), sensori integrati, ADC, comparator

    Applicazioni: IoT, sensori, dispositivi medici portatili, energy harvesting

    Popolarità: molto popolare in applicazioni a basso consumo

    Infineon XC800

    Serie: XC800, XC800 Safety, XC800 Comfort

    Caratteristiche: microcontroller automotive, sicurezza funzionale, CAN-FD, sicurezza integrata ASIL-B/D

    Applicazioni: sistemi di sicurezza automotive, controllo motore, body control

    Popolarità: emergente, focus su automotive safety

    Progetti Implementati

    MIDI Entry

    Obiettivo: Implementare un'interfaccia MIDI a 5 pin per generare note musicali

    Hardware: PIC18F2520, potenziometro per transpose, display LCD 16x2, buzzer

    Software: decoder note MIDI, gestione polifonica a 16 canali, gestione interrupt UART

    Caratteristiche: 4 pollici per ottava, sustain, pitch bend, controllo di modulazione

    Applicazioni: strumenti musicali digitali, controller MIDI per software

    Controllo Motore Lavatrice

    Obiettivo: Sviluppare un sistema di controllo per motore asincrono monofase con PWM per velocità

    Hardware: PIC16F877A, driver MOSFET IRFZ44N, sensore Hall, potenziometro

    Software: PWM a 10-bit, controllo closed-loop per velocità, protezione sovraccarico

    Caratteristiche: controllo velocità variabile, avvio graduale, protezione da sovraccarico, indicatori LED

    Applicazioni: lavatrici domestiche, lavaggio industriale, controllo velocità motori

    Forno Mono-Zona

    Obiettivo: Sistema di controllo temperatura per forno elettrico monozona con PID

    Hardware: PIC18F2520, termistore NTC, relay driver, display LCD, pulsanti

    Software: algoritmo PID, gestione temperatura di sicurezza, controllo relay, logging temperatura

    Caratteristiche: impostazione temperatura, modalità manuale/auto, allarme temperatura alta, protezione surriscaldamento

    Applicazioni: forni industriali, forni domestici, controllo temperatura in generale

    Termostato CNT

    Obiettivo: Sistema di misurazione temperatura con termocoppia e visualizzazione

    Hardware: PIC16F1789, termocoppia K-type, ADC, display LCD, sensore umidità

    Software: misurazione temperatura, compensazione umidità, allarme temperatura, visualizzazione in tempo reale

    Caratteristiche: misurazione ±0.5°C, allarme +100°C/-50°C, visualizzazione °C/F, memoria dati

    Applicazioni: monitoraggio temperatura ambiente, controllo HVAC, monitoraggio industriale

    Applicazioni OS-based

    FreeRTOS: Sviluppo di un sistema operativo real-time per PIC32MX

    ChibiOS: RTOS leggero per microcontroller 8-bit, sensori, comunicazione

    Bare-metal: Bootloader USB, programming in-circuit, gestione firmware

    Caratteristiche: scheduler real-time, gestione dei task, semafori, code messaggi

    Applicazioni: sistemi embedded real-time, controllo di automazione, comunicazione periferica
